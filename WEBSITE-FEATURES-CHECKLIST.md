## 1. ESSENTIAL PAGES & SECTIONS

### Homepage (Single Most Important Page)

- [ ] **Instant Quote Widget** - Above-the-fold form with: ZIP code, event date, guest count, event type

- [ ] **Blog Preview / Educational Content** - 2-3 recent articles or guides
- [ ] **Final CTA Section** - Full-width banner with phone number, quote button, and urgency messaging

## 4. SEO REQUIREMENTS

### Technical SEO
- [ ] **SSL/HTTPS** - Entire site secured
- [ ] **XML Sitemap** - Submitted to Google Search Console
- [ ] **Robots.txt** - Properly configured
- [ ] **Canonical URLs** - Prevent duplicate content
- [ ] **301 Redirects** - For any URL changes
- [ ] **404 Page** - Custom branded error page with navigation
- [ ] **Mobile-First Indexing** - Fully responsive, mobile-optimized
- [ ] **Page Speed Optimization** - LCP < 2.5s, CLS < 0.1, INP < 200ms

### On-Page SEO
- [ ] **H1 Tags** - One per page, keyword-focused (e.g., "Luxury Restroom Trailer Rental Long Island NY")
- [ ] **Meta Titles** - 50-60 characters, keyword + brand
- [ ] **Meta Descriptions** - 150-160 characters, compelling with CTA
- [ ] **Header Hierarchy** - Proper H1 > H2 > H3 nesting
- [ ] **Image Alt Text** - Descriptive, keyword-rich (format: "Business Name - Event Type - Location")
- [ ] **Internal Linking** - Cross-link between related pages
- [ ] **URL Structure** - Clean, descriptive slugs (/luxury-restroom-trailers/, /wedding-restroom-rental/)

### Local SEO
- [ ] **Google Business Profile** - Fully optimized with photos, hours, services, Q&A
- [ ] **NAP Consistency** - Name, Address, Phone identical across all citations
- [ ] **LocalBusiness Schema** - JSON-LD structured data with:
  - @type: LocalBusiness (or more specific: "RentalAgency")
  - name, address, telephone, openingHours
  - geo coordinates (latitude, longitude)
  - areaServed (list all service cities)
  - priceRange
  - aggregateRating (if eligible)
  - sameAs (social media links)
- [ ] **Service Schema** - For each trailer type
- [ ] **FAQ Schema** - On FAQ page for rich results
- [ ] **Event Schema** - If hosting or attending expos/bridal shows
- [ ] **BreadcrumbList Schema** - For navigation hierarchy
- [ ] **Google Maps Embed** - On contact page
- [ ] **City-Specific Landing Pages** - For each major service area
- [ ] **Local Keywords** - "luxury restroom trailer rental [City]" on each location page
- [ ] **Local Citations** - Yelp, BBB, wedding directories, local business directories
- [ ] **Review Generation Strategy** - Automated post-event review requests

### Content SEO
- [ ] **Blog with Regular Posts** - 2-4 posts per month
- [ ] **Long-Form Guides** - 2,000+ word comprehensive articles
- [ ] **Target Keywords:**
  - "luxury restroom trailer rental Long Island"
  - "portable restroom rental NYC"
  - "wedding restroom trailer rental"
  - "luxury porta potty rental near me"

## 7. ACCESSIBILITY REQUIREMENTS (WCAG 2.1 AA)

### Visual Accessibility
- [ ] **Color Contrast Ratio** - Minimum 4.5:1 for normal text, 3:1 for large text
- [ ] **Don't Rely on Color Alone** - Use icons, text, patterns alongside color
- [ ] **Resizable Text** - Up to 200% without content loss
- [ ] **Focus Indicators** - Visible outlines on all interactive elements
- [ ] **Alt Text on All Images** - Descriptive, meaningful
- [ ] **No Autoplay Media** - Or provide pause/stop controls

### Navigation Accessibility
- [ ] **Keyboard Navigation** - Full site usable without mouse
- [ ] **Skip Navigation Link** - "Skip to main content" for screen readers
- [ ] **ARIA Labels** - On all interactive elements
- [ ] **Logical Tab Order** - Natural flow through page elements
- [ ] **Focus Management** - After modal opens/closes, form submissions
- [ ] **Breadcrumb Navigation** - Clear location indicator

### Form Accessibility
- [ ] **Label Every Input** - Visible labels (not just placeholders)
- [ ] **Error Messages** - Clear, specific, announced to screen readers
- [ ] **Required Field Indicators** - Visual and programmatic
- [ ] **Input Autocomplete Attributes** - Help autofill (name, email, tel, address)
- [ ] **Fieldset/Legend for Groups** - Radio buttons, checkboxes grouped properly

### Content Accessibility
- [ ] **Proper Heading Hierarchy** - H1 > H2 > H3 without skipping levels
- [ ] **Readable Font Size** - Minimum 16px body text
- [ ] **Line Spacing** - Minimum 1.5x line height for body text
- [ ] **Plain Language** - 8th grade reading level for main content
- [ ] **Video Captions** - All videos captioned
- [ ] **PDF Accessibility** - Tagged PDFs if offering downloadable content
- [ ] **Accessibility Widget** - UserWay or similar overlay tool (Privy Chambers uses this)
- [ ] **Accessibility Statement Page** - Document commitment and contact for issues

## 8. PERFORMANCE REQUIREMENTS

### Core Web Vitals Targets
- [ ] **LCP (Largest Contentful Paint)** - Under 2.5 seconds
- [ ] **CLS (Cumulative Layout Shift)** - Under 0.1
- [ ] **INP (Interaction to Next Paint)** - Under 200 milliseconds
- [ ] **TTFB (Time to First Byte)** - Under 600 milliseconds

### Image Optimization
- [ ] **WebP/AVIF Format** - Modern compressed formats
- [ ] **Responsive Images** - srcset with multiple sizes
- [ ] **Lazy Loading** - Below-the-fold images load on scroll
- [ ] **Image Compression** - Max 200KB per image, hero images max 500KB
- [ ] **Explicit Dimensions** - Width/height attributes to prevent CLS

### Code Optimization
- [ ] **Minified CSS/JS** - Production builds compressed
- [ ] **Code Splitting** - Load only needed JavaScript per page
- [ ] **Tree Shaking** - Remove unused code
- [ ] **Critical CSS Inline** - Above-fold styles in <head>
- [ ] **Deferred JavaScript** - Non-critical scripts loaded async/defer
- [ ] **Font Optimization** - Preload key fonts, font-display: swap
- [ ] **CDN Delivery** - Static assets served from edge locations

### Hosting & Infrastructure
- [ ] **Fast Hosting** - VPS or cloud, not shared hosting
- [ ] **SSL/TLS** - HTTPS with modern cipher suites
- [ ] **HTTP/2 or HTTP/3** - Multiplexed connections
- [ ] **Gzip/Brotli Compression** - Server-side text compression
- [ ] **Browser Caching** - Appropriate cache headers for static assets
- [ ] **Service Worker** - Offline capability for repeat visitors (optional, advanced)
